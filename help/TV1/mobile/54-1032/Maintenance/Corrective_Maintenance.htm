Corrective Maintenance

If a system component fails, it will need to be replaced by a KVH-certified technician. Instructions are provided with the replacement part.

Important!

Field replaceable units should only be replaced by a trained, KVH-certified technician. To find a technician near you, visit www.kvh.com/wheretogetservice or contact KVH Technical Support (see Technical Support).
