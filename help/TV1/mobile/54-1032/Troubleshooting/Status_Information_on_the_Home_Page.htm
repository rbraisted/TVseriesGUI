Status Information on the Home Page

The Home page of the web interface provides basic system status information, as described below.

Home Page

Item

Description

A

Three status indicators match the physical status lights on the TV-Hub (see TV-Hub Status Indicators)

B

Current satellite switching mode: Automatic or Manual

C

Currently selected satellite, satellite signal strength, and general status of the antenna:

•   Tracking

•   Initializing

•   Searching

•   Gyro Calibration

•   Idle

•   Error

D

Currently selected master receiver (only shown in Automatic satellite switching mode)

E

Antenna model

F

Latitude/longitude of your current location, based on GPS input or manual entry

G

List of satellites currently installed in the antenna

H

Your current true heading (requires NMEA input)

I

Tracking status visual display; a green line between the antenna and the satellite indicates the antenna is tracking the satellite
