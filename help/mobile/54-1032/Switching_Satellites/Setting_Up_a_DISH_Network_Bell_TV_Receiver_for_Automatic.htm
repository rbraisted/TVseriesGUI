Setting Up a DISH Network/Bell TV Receiver for Automatic Switching

For automatic switching to work properly with your DISH Network or Bell TV receivers, be sure to observe the following requirements:

  • Make sure the receiver is DISH pro-compatible. Look for the DISH Pro logo on the box.

  • Select a preset satellite group (see Selecting a Satellite Group). User-defined groups are limited to manual satellite switching only.

  • Run the TracVision Setup Wizard and initiate a Check Switch test on the receiver when prompted to do so. See Running a Check Switch Test on a DISH Network or Bell TV Receiver for details. You can launch the Setup Wizard from the web interface (go to Settings > General Settings).
