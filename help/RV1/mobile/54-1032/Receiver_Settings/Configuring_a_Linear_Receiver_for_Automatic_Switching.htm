Configuring a Linear Receiver for Automatic Switching

For automatic satellite switching to work properly using the DiSEqC communication protocol, you need to set up your receivers for the same satellites you installed in the TracVision antenna. Be sure to set up the satellites in the receiver in the exact same order as they were set up in the antenna. The specific setup process varies among receiver models – refer to your receiver owner’s manual for details. Use the table and figure below as a guide.
