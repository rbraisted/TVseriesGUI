Setting Up a DIRECTV Receiver for Automatic Switching

During system installation, your SWM-compatible receivers should have been set up for you. (See the Installation Guide for details.) If you wish to add a new SWM-compatible receiver, follow these steps:

  1. Set the receiver’s Dish Type to 3-LNB and Switch Type to SWM. See Setting the Dish Type on a DIRECTV Receiver for details.

  2. Set up the receiver to allow external device access. See Allowing External Access on a DIRECTV Receiver for details.

  3. Add the new receiver to the Autoswitch page of the web interface. Enter the receiver’s IP address and assign it a friendly name (e.g., “Salon”). See Finding the IP Address of a DIRECTV Receiver for details on how to find the receiver’s IP address.
