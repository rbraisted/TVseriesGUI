Using an iPhone/iPod touch to Update Software

To update the software using an iPhone/iPod touch, follow these steps:

  1. If you haven’t already done so, download the mini-VSAT app from the Apple iTunes® App Store.

  2. Use the iPhone/iPod touch app to download the latest software version to your iPhone/iPod touch at any time.

  3. Make sure the TracPhone system is turned on and initialized (the ICM front panel LCD shows “Searching” or “Tracking”).

  4. Connect your iPhone/iPod touch to the TracPhone system via its wireless connection, then use the app to install the software.
