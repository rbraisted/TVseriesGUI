Software Updates

KVH periodically releases new versions of software to enhance the performance and capabilities of the system. You can update your system to this latest software at any time by downloading the file from KVH’s Update server and uploading it to the ICM. When you update the software, the new version simply overwrites the previously installed version.

A single software package includes updates to all system components in the antenna and ICM, with the exception of the modem. Modem updates are downloaded automatically via the satellite connection. When a modem update is in progress, the TracPhone V11-IP web interface shows “Updating” under Connection Status, and the ICM front panel display shows “Do not power down - Updating” for 30 minutes.

There are several methods to choose from to update the software:

  • Using the ICM to Update Software

  • Using Your Computer to Update Software

  • Using an iPhone/iPod touch to Update Software

  • Using a USB Flash Drive to Update Software
