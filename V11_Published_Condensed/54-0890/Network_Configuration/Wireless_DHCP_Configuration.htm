Wireless DHCP Configuration

In a wireless (Wi-Fi) DHCP configuration, vessel computers connect to the ICM via its built-in wireless access point (WAP) and receive IP addresses from the ICM.

Important!

The range of the ICM’s WAP will depend on the layout and structure of the vessel. For example, wireless signals degrade when passing through bulkheads and near metal masses. If the system is installed on a steel vessel, you might need a special WAP and the services of a technician with advanced networking expertise.

Wireless DHCP Configuration (Example)

 

To configure a wireless DHCP network connection, follow these steps:

  1. Make sure the computers are equipped with a wireless network interface card and configured for DHCP addressing (see Configuring Computers for DHCP).

  2. Enable the wireless access point on the ICM and apply security settings to protect it from outside intrusion (see Wireless Settings on ICM).
