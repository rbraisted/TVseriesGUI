CommBox Settings on ICM

If the optional CommBox software is enabled, the ICM’s network settings are configured by KVH over the air. The Network Settings page of the ICM will not contain any configurable fields.

NOTE: See CommBox Features for details on enabling the CommBox software.

For complete details about the CommBox software, refer to the CommBox manual. To view the CommBox manual, follow these steps:

  1. At the TracPhone V11-IP web interface, click the CommBox tab.

    2. CommBox Tab

     

  2. Click Log in to CommBox Control.

  3. Log into the CommBox web page with your CommBox user name and password.

    4. NOTE: The CommBox user name and password differs from the ICM’s Administrator password.

  4. At the CommBox web page, click the Help tab.
