About this Help System

This Help system provides basic operation, configuration, troubleshooting, and maintenance information for the TracPhone V11-IP system. If you have a question about the system, you will likely find the answer in the Help.

Important!

You might need to clear your web browser’s cache in order to view the latest version. Refer to Clearing the Cache of Your Web Browser for details.

This Help system is compatible with the following web browsers: Internet Explorer, Mozilla Firefox, Google Chrome, and Apple Safari. If you are using Internet Explorer version 10, you will first need to adjust the width of the Help’s navigation pane to view topics due to a known software issue.

You can choose between two versions of Help:

Condensed Help

  • Resides on the ICM locally

  • Text-only format

Extended Help

  • Resides on the KVH website (airtime charges may apply)

  • Media-rich format, complete with images and tutorials

To access either of these resources from the ICM, follow these steps:

  1. At the TracPhone V11-IP web interface, click the Support tab.

  2. On the Help menu, click the desired option.
