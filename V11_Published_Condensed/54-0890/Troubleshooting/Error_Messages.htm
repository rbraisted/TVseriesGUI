Error Messages

This section lists the error message codes that might appear on the TracPhone V11-IP web interface and front panel LCD to indicate a system problem. Many of these faults should only be repaired by a KVH-certified technician; contact KVH Technical Support for assistance (see Technical Support).

Error

Description

101

There is a short circuit in the antenna data cable. Check the cable.

102

The ICM is not providing the proper DC power to the antenna (this DC voltage must be present to enable the antenna’s AC power circuit). Check the wires on the back of the ICM.

103

A software update file was not found on the USB device. Try these steps:

•   Remove the USB device from the ICM, wait 10 seconds, then reinsert it.

•   Verify that a valid TracPhone V11-IP update file (ending in “.kvh”) is present on the USB device.

•   Replace the update file on the USB device with a new file.

•   Use a different USB device.

104

The software update file you selected is corrupt. Try a new update file.

106

The ICM has lost communications with the antenna. Try these steps:

•   Check the vessel AC power connection to the antenna.

•   Check the wiring on the back of the ICM.

•   Check the antenna data cable.

•   Try turning the system off, then back on.

107

The ICM’s internal temperature exceeds acceptable limits. Try these steps:

•   Listen for the cooling fan inside the ICM. If you don’t hear the fan, contact KVH.

•   If you hear the cooling fan, turn off the system and allow it to cool down. Then vacuum the ICM’s vents (see Preventive Maintenance).

•   If the problem persists, you might need to relocate the unit to an area that provides better ventilation.

111

The ICM is unable to initialize its built-in wireless access point (WAP). Contact KVH.

112

There is an internal fault in the ICM. Contact KVH.

114

The GPRS modem (remote service & support module) inside the antenna is not receiving any power. Check the data cable and the wires on the back of the ICM.

116

The ICM is not providing the proper DC power to the GPRS modem (remote service & support module) inside the antenna. Check the wires on the back of the ICM.

117

There is an open circuit in the BUC power wiring, or the antenna’s BUC might be faulty – contact KVH.

118

There is a short circuit in the BUC power wiring, or the antenna’s BUC might be faulty – contact KVH.

120

The internal temperature of the ICM’s built-in modem exceeds acceptable limits. Try these steps:

•   Listen for the cooling fan inside the ICM. If you don’t hear the fan, contact KVH.

•   If you hear the cooling fan, turn off the system and allow it to cool down. Then vacuum the ICM’s vents (see Preventive Maintenance).

•   If the problem persists, you might need to relocate the unit to an area that provides better ventilation.

121

The ICM is not receiving satellite signal strength (RSSI) data from its built-in modem. Try these steps:

•   Turn the ICM off, then back on.

•   Check the serial data jumper cable on the back of the ICM connecting between A1 and A2.

122

The ICM’s built-in modem is not communicating. Try these steps:

•   Check the ICM jumper cables.

•   Try turning the ICM off, then back on.

•   If the error persists, contact KVH.

123

The ICM’s built-in modem is not responding to status queries. Contact KVH.

124

The software update file you selected is not valid for the TracPhone V11-IP. Try a new update file.

125

The ICM has lost communications with the GPRS modem (remote service & support module) inside the antenna. Check the data cable and the wires on the back of the ICM.

127

The antenna is not accepting the no-transmit zones setup from the ICM. Contact KVH.

128

The software update failed. Retry updating the software. If it fails again, check the data cable and the wires on the back of the ICM.

200

The antenna is not receiving adequate DC power from the ICM (this DC voltage must be present to enable the antenna’s AC power circuit). Check the data cable and the wires on the back of the ICM.

202

The antenna’s main board has lost communications with the skew and cross-azimuth motors. Contact KVH.

The antenna’s main board has lost communications with the azimuth and elevation motors. Contact KVH.

The antenna’s main board has lost RF communications. Try updating the software (see Software Updates).

The antenna’s azimuth motor has a power failure. Contact KVH.

The antenna’s azimuth motor or limit switch failed. Contact KVH.

265
