Event Log

The Event Log keeps a historical record of the error messages the antenna system generates. The most recent entries in this log are displayed on the Support page of the TracPhone V11-IP web interface.

To view the complete Event Log, follow these steps:

  1. At the TracPhone V11-IP web interface, click the Support tab.

  2. On the System Logs menu, click Event Log.
